1. Xoa hoan toan thu muc images - thu muc nay danh dau tung step agent da di qua, neu khong xoa code se ghi de len va anh huong luong chay
2. Dam bao dang o dung duong dan goc cua thu muc
[...]\RL-Honeypot

-------------------------------------------
Neu su dung Linux:
3. Khoi tao mot moi truong ao moi
python -m venv .
4. Cai dat cac package can thiet
pip install -r requirements.txt


Neu su dung Windows:
3. Activate môi trường ảo bằng lệnh
.\Script\activate
--------------------------------------------

5. Chay thu muc tong hop NetworkHoneypotEnv.py trong folder Development
python .\Development\NetworkHoneypotEnv.py

Do mo hinh nay su dung tensorflow cho CPU nen thoi gian chay kha lau. Day la thieu sot ma nhom can khac phuc trong tuong lai.
Trong phan demo nay de don gian va de quan sat luong chay nhom xin duoc dua toan bo code vao cung mot file NetworkHoneypotEnv.py. 
Trong tuong lai code can duoc chia thanh nhieu file khac nhau de de quan ly hon.
